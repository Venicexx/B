# Claude Code 能做什么？

我是 **Claude Code**，一个运行在终端里的 AI 编程助手。以下是我的核心能力。

---

## 1. 代码编写与编辑

- **写新文件** — 根据需求创建完整的代码文件（任何语言）
- **精确编辑** — 对现有文件进行精确的字符串替换，不会破坏其他部分
- **批量修改** — 一次性替换文件中所有匹配的内容

## 2. 阅读与理解代码

- **读取文件** — 查看任何文件的内容
- **搜索代码** — 用正则表达式在项目中搜索特定模式
- **文件匹配** — 用 glob 模式快速找到文件（如 `**/*.tsx`）
- **理解项目结构** — 浏览目录、分析代码架构

## 3. 终端命令执行

- **运行命令** — 通过 PowerShell 执行各类终端命令
- **Git 操作** — 提交、分支管理、查看日志、合并等
- **包管理** — npm、pip、cargo 等包管理器的安装和更新
- **后台任务** — 长时间运行的任务可以放到后台，完成后通知你

## 4. 调试与问题排查

- **阅读错误日志** — 理解报错信息并定位问题
- **分析 bug** — 根据代码逻辑推断问题根源
- **修复建议** — 提供具体的修复方案并直接实施
- **测试运行** — 执行测试套件，查看测试结果

## 5. 代码审查

- **Code Review** — 检查代码中的 bug、安全漏洞、性能问题
- **简化重构** — 发现可以复用、简化、优化的代码
- **安全审查** — 专门针对安全漏洞的审查

## 6. 项目管理

- **任务列表** — 创建和跟踪待办事项，组织复杂的多步骤工作
- **工作计划** — 进入计划模式，先设计方案再动手写代码
- **Git 工作树** — 在隔离的环境中工作，不影响主分支

## 7. 网络搜索与调研

- **网页搜索** — 搜索最新技术文档、解决方案
- **网页抓取** — 读取网页内容并根据你的问题提取答案
- **深度调研** — 多源搜索、交叉验证、生成带引用的研究报告

## 8. 设计与 UI

- **设计系统同步** — 将本地组件库同步到 Claude Design 项目
- **前端组件** — 编写 HTML/CSS/React/Vue 等各类 UI 代码

## 9. 定时任务

- **定时提醒** — 设置一次性或重复的定时任务
- **后台轮询** — 定期检查某个状态（如部署进度）
- **持久化任务** — 即使重启也能保留的定时任务

## 10. 记忆功能

- **记住偏好** — 我会记住你的偏好、项目约定和重要信息
- **跨会话记忆** — 下次对话时我会回忆起之前的约定

## 11. 集成与扩展

- **MCP 工具** — 支持通过 MCP 协议连接外部工具和服务
- **自定义技能** — 可配置项目级的自定义命令和工作流
- **Hook 钩子** — 在特定事件（如保存文件、启动会话）时自动执行脚本

## 12. 多 Agent 协作

- **并行处理** — 同时启动多个 Agent 处理不同的子任务
- **工作流编排** — 设计复杂的多阶段、多 Agent 协作流程
- **专项 Agent** — 不同类型的 Agent 擅长不同领域（探索、计划、审查等）

---

## 我能处理的文件类型

- 几乎**所有编程语言**的源代码
- **配置文件**（JSON、YAML、TOML、XML 等）
- **Markdown 文档**
- **Jupyter Notebook**（.ipynb）
- **PDF 文件**（阅读）
- **图片文件**（查看 PNG、JPG 等）

---

## 我不能做什么

- 我不能**登录你的账户**或代表你执行需要交互式认证的操作
- 我不能在**没有明确授权**的情况下执行破坏性操作（删除、强制推送等）
- 我不能访问**互联网上的私有/认证页面**
- 我不是万能的 — 遇到不确定的事情我会先和你确认

---

> 💡 **小提示**：直接告诉我你想做什么，我会选择最合适的方式来帮你完成。如果你不确定怎么做，也可以先问我有什么方案。
