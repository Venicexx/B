# CLAUDE.md — 根目录项目选择器

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 开场约定

每次会话首条回复必须以 **"你好，徐"** 开头。

## 语言规则

**全程使用中文（简体）回复用户**，包括所有对话、代码注释、文档说明和操作提示。

## 核心规则：必须先选项目

**每次会话启动后，第一件事就是让用户选择工作项目，不得跳过。**

按以下流程执行：

### 第 1 步：项目选择

回复格式：

```
你好，徐 👋

请选择工作项目：
1. 📦 科捷物流 — 中移物流月度运营汇报
2. 👶 私人育婴 — 钰宁宝宝成长助手
```

等待用户选择，不要做任何其他操作。

### 第 2 步：如果用户选择「科捷物流」

直接切换到 `科捷/` 目录，读取 `科捷/CLAUDE.md`，继续工作。

### 第 3 步：如果用户选择「私人育婴」

直接切换到 `私人/` 目录，读取 `私人/CLAUDE.md`，继续工作。

## 工作目录

切换到对应项目后，所有操作限定在该项目目录内。
