# 增值服务流水线作业SOP设计文档

> 操作手册与产品使用说明书置换 · 流水线作业指导卡

---

## 1. 项目概述

### 1.1 背景

北京科捷物流广东省仓库承接中移物流的增值服务（VAS）业务，需要将原装箱中每台设备内的操作手册和产品使用说明书置换为新版本。为规范流水线作业、确保质量和效率，需要制作一套标准作业指导卡（SOP）。

### 1.2 操作流程（7个标准动作）

| 步骤 | 动作描述 | 关键注意事项 |
|:----:|---------|-------------|
| ① | 用刀划开原装箱 | 避免划伤内部销售包装 |
| ② | 取出一台销售包装的货物 | 轻拿轻放 |
| ③ | 用小刀划开销售包装的封口贴 | 沿封口线划开，不损坏包装 |
| ④ | 打开销售包装，取出原有的操作手册（设备上方）和产品使用说明书（设备下方） | **注意保留合格证（三角形）** |
| ⑤ | 将新版操作手册和产品使用说明书放入销售包装内 | 位置对应：操作手册在上，说明书在下 |
| ⑥ | 还原销售包装并贴上封口贴 | 封口贴须贴正、贴牢 |
| ⑦ | 将已置换好的货物装回箱中 | 摆放整齐 |

### 1.3 作业参数

| 参数 | 值 |
|------|-----|
| 每箱设备数 | 20台 |
| 流水线模式 | 按人数分配工位 |
| 人员版本 | 1人版、2人版、3人版、4人版 |

## 2. 输出物格式

### 2.1 输出形式

- **格式**：A4横向（297×210mm）海报/看板
- **载体**：每版本一张独立海报，打印张贴于流水线旁或管理看板
- **总数量**：4张（1/2/3/4人版各一张）

### 2.2 海报版面结构

```
┌─────────────────────────────────────────────────┐
│  顶栏：操作名称 + 版本标识 + 客户/地点/参数      │
├──────────────┬──────────────────────────────────┤
│  矩阵区      │  参考图区                         │
│  (步骤×工位) │  (4张操作参考图)                 │
│  行=7个步骤  │  2×2排列，每张标步骤编号         │
│  列=工位A/B… │  合格证图旁加"⚠️勿遗失"标注     │
│  交叉=✅负责  │                                  │
├──────────────┴──────────────────────────────────┤
│  底栏：作业准备（工具清单）+ 质量标准 + 版本信息  │
└─────────────────────────────────────────────────┘
```

## 3. 工位动作分配方案

### 3.1 分配原则

- 按动作数量平均分配（非工时分配）
- 保持动作逻辑连贯性（不拆散同一单元的关联动作）
- 传递方向一致：A→B→C→D

## 4. 质量标准（所有版本通用）

### 4.1 质量红线

1. **不得损坏销售包装** — 划封口时沿封口线操作，力度适中
2. **不得遗失合格证** — 取旧手册时注意保留包装内的三角形合格证，不可丢弃或混入废纸
3. **封口贴须贴正、贴牢** — 位置与原封口贴一致，确保平整无气泡
4. **新旧手册不得混淆** — 取出旧手册后立即与替换废料分开放置

### 4.2 作业准备（工具清单）

- 美工刀（锋利，备用刀片）
- 封口贴（备用量≥当日产能）
- 新操作手册 x 20/箱
- 新产品使用说明书 x 20/箱
- 废料回收箱（旧手册/说明书）

## 5. 技术实现方案

### 5.1 实现方式

输出物为 **Word文档** 或 **可打印PDF**，每个版本一页A4横向。

建议使用以下方式之一制作：
- **选项A（推荐）**：使用 Python + python-docx 生成包含矩阵表格和插入图片的 .docx 文件，转为PDF打印
- **选项B**：使用 Excel 手工制作A4横向表格打印
- **选项C**：使用 PPT 一张幻灯片一页，设置A4横向尺寸

### 5.2 自动化生成（长期方案）

如果后续需要反复调整或新增版本，可编写 Python 脚本：
- 输入参数：版本人数（1-4）、每箱台数
- 自动绘制矩阵表格 + 插入对应✅标记
- 自动插入4张参考图到指定位置
- 输出 A4横向 PDF
